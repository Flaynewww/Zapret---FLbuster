┌─────────────────────────────────────────────────────────────┐
│ ШАГ 1: ЗАПУСК                                               │
└─────────────────────────────────────────────────────────────┘
   Разархивируйте в архиве папку FLbuster На рабочий стол.
   1. Найдите FLbuster.exe
   2. Правой кнопкой → "Запуск от имени администратора"
   3. Подтвердите запрос UAC

   ⚠️ ВАЖНО: Обязательно от администратора!

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 2: УСТАНОВКА СЛУЖБЫ                                     │
└─────────────────────────────────────────────────────────────┘

   1. Перейдите на вкладку "⚙️ Службы"
   2. Выберите стратегию (рекомендуется: general (ALT).bat)
   3. Нажмите "🧪 Test Strategy" для проверки (опционально)
   4. Нажмите "📥 Install Selected Strategy"
   5. Дождитесь сообщения об успехе

┌─────────────────────────────────────────────────────────────┐
│ ШАГ 3: ПРОВЕРКА                                             │
└─────────────────────────────────────────────────────────────┘

   1. Перейдите на вкладку "🏠 Главная"
   2. Проверьте статусы:
      ✅ Service Status: Running (зеленый)
      ✅ Bypass Status: Active (зеленый)
   3. Откройте Discord или YouTube
   4. Проверьте работу

   🎉 ГОТОВО! Теперь все работает!
